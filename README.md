# Mechatronics Snake Robot

Final project developed for the Mechatronics course in the Robotics Engineering degree.

## Description

This project consists of the design and implementation of a modular robotic system using mechanical design, electronics, and Arduino programming.

The robot was designed using FreeCAD and controlled with Arduino.

The project includes:

- Mechanical design of the robot
- CAD models
- Arduino control program
- Assembly structure
